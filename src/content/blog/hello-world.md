歡迎來到我的博客！這是我的第一篇文章，作為程序員，當然要從"Hello World"開始。

## 為什麼開始寫博客？

寫博客對我來説是一種思考和分享的方式。通過寫作，我可以：

1. 整理自己的知識體系
2. 分享技術心得與發現
3. 記錄學習歷程
4. 與更多志同道合的開發者交流

## 代碼示例

這是一個簡單的 JavaScript 函數：

```javascript
function greeting(name) {
  return `Hello, ${name}!`;
}

// 調用函數
console.log(greeting("World")); // 輸出: Hello, World!
```
